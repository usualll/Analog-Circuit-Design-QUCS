# Analog Circuit Design & System Integration Portfolio

## Overview
This repository contains a comprehensive suite of transistor-level analog circuit simulations and a complete system-level capstone project developed during the **Texas Instruments 'BYTE' Technical Program**. 

The objective of this portfolio is to demonstrate the bridge between theoretical solid-state physics and practical electrical engineering by modeling, designing, and verifying analog circuits and sensor systems using the open-source **Quite Universal Circuit Simulator (QUCS)**.

## Technical Competencies Demonstrated
* **System-Level Integration:** Designing front-end analog interfaces to process physical environmental inputs (sensor interfacing and signal conditioning).
* **Circuit Analysis:** Executing DC bias point calculations, Transient (time-domain) analysis, and AC Sweep (frequency response/Bode plots) analysis.
* **Device Physics Application:** Modeling the real-world behavior of passive networks, diodes, BJTs, and MOSFETs, translating theoretical semiconductor physics into verifiable circuit behavior.

## Repository Structure
The project files and technical reports are organized by progressive complexity, culminating in a practical hardware system:

* **Week 00:** Software Familiarization & EDA Environment Setup
* **Week 01:** DC Operating Point Analysis 
* **Week 02:** Transient Response of Passive Networks (RC/RL)
* **Week 03:** AC Sweep and Frequency Response Analysis
* **Week 04:** Diode Modeling and Rectification Circuits
* **Week 05:** Transistor I-V Characteristics (BJT & MOSFET)
* **Week 06:** Analog Amplifier Design & Performance Extraction
* **Capstone Project:** Smart Flood Detection System

## 🚀 Featured Capstone: Smart Flood Detection System
To culminate the TI BYTE training, we designed and simulated a complete, applied electronic system. This project transitioned my focus from isolated circuit analysis to **system-level integration**.

**Key Engineering Highlights:**
* **Sensor Interfacing:** Designed the front-end analog interface to process physical environmental inputs.
* **Signal Conditioning & Logic:** Utilized analog comparators/amplifiers to establish precise voltage thresholds for water level detection.
* **System Design:** Integrated discrete analog components to create a reliable, automated alert mechanism without relying on complex microcontrollers.
